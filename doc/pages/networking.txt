/**
  \defgroup networking Networking
  
  \section Overview 
  The Make Controller comes with a fully loaded network system, including:
  - TCP sockets
  - TCP server sockets
  - UDP sockets
  - DHCP
  - DNS name resolution
  
  On top of these, additional services are provided:
  - simple web server
  - web client
  
  \section Breakdown
  
  Check the WebClient class if you want your board to head out onto the internet and 
  connect to other sites or services.
  
  Check the WebServer class to provide access to your board from the internet or from a web browser.
  
  Check the TcpSocket, TcpServer and UdpSocket classes to make socket level connections.
  
  Check the Network class to configure your board's network settings (address, gateway, dhcp, dns).
*/
